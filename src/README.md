# real-estate

Vue.js movie database 

## features

- themoviedb.org API 
- View popular movies
- View movie details, cast, crew and trailer
- Search movies
- Infinite scroll
- Dynamic background based off hero image colors
(source palette extraction algorithm: https://github.com/zygisS22/color-palette-extraction)

## tech

- Vue.js
- Typescript
- Pinia
- Vue Router
